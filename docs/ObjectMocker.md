# ObjectMocker

> 对象生成

在描述一个对象的时候，这个对象的数据规范其实就已经出现了。因此，描述对象后，可以按照描述生成对象的数据。
`GOM` 中，支持 `JSON` 的全部数据类型：

- `数字` (`number`): `123`, `123.123`
- `布尔值` (`bool`): `true`, `false`
- `字符串` (`string`): `"字符串"`
- `结构数组` (`array`) `["字符串1", "字符串2"]`， `[123, 234]`
- `对象` (`object`) `{"key": "value"}`

您在使用过程中可以通过描述一个字段的类型来获取一个完整的对象内容。
